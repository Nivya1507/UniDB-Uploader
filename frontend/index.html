OOP Concept Architecture

Encapsulation

All user records maintain protected/private base data accessed only via public interface methods (getters/setters).

Inheritance

Student, Professor, and Admin derived classes inherit from a unified Person base class.

Polymorphism

Dynamic binding on virtual methods (toJSON, toCSV, getRole) handles objects polymorphically on disk/JSON operations.

Persistence

C++ File IO manages direct serialization and deserialization of the object structures to persistent CSV databases.
